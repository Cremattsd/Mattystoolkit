Matty’s Toolkit Light – Privacy Policy

Last Updated: November 2025

Matty’s Toolkit Light is a lightweight Chrome extension built solely to enhance workflow inside RealNex by enabling optional split-view routing for Contacts, Companies, and Properties. The extension does not collect, store, transmit, process, or share any user data.

Information We Collect

No data of any kind is collected.
The extension does not collect:

  • Personal information
  • Financial or payment information
  • Authentication or login credentials
  • Browsing history
  • Communications or messages
  • Location data
  • Website content or page text

How the Extension Works

The extension only performs local tab routing logic. It detects which RealNex link was clicked and may open or update a dedicated browser tab. All logic runs locally inside the browser and no information leaves the user’s device.

Permissions Explanation

tabs: Used exclusively to open or update the split-view tabs.

storage: Stores only ON/OFF toggle values for user preferences.

scripting: Injects the content script so link-click detection works.

Host Permission (realnex.com): Required only to detect when a user clicks a RealNex record link. No data is scraped or collected.

Data Sharing

Matty’s Toolkit Light does not share, transfer, sell, or transmit any data to any third party.

Remote Code

No remote code is used. All code is packaged entirely within the extension.
